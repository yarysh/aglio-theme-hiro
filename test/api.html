Real World API

This API Blueprint demonstrates a real world example documenting a portion of App.net API.

  • Posts

Posts

This section groups App.net post resources.
GET /stream/0/posts/{post_id}

Returns a specific Post.

  • Parameters
  • Name Description Required Type Example
    post_id

    The id of the Post.

    		 Required string 1
  • Response 200 SHOW

    "data": {
        "id": "1", // note this is a string

        "source": {
            "client_id": "udxGzAVBdXwGtkHmvswR5MbMEeVnq6n4",

DELETE /stream/0/posts/{post_id}

Delete a Post. The current user must be the same user who created the Post.

  • Parameters
  • Name Description Required Type Example
    post_id

    The id of the Post.

    		 Required string 1
  • Response 204 SHOW
